Завдання 3 — висновки

Було порівняно три алгоритми пошуку підрядка:

- Боєр–Мур
- КМП (Кнута–Морріса–Пратта)
- Рабіна–Карпа

Пошук виконувався у двох текстових файлах. Для кожного тексту перевіряли:

- підрядок, який є у тексті
- підрядок, якого немає у тексті

Результат

У всіх випадках:

1. КМП — найшвидший алгоритм**
2. Боєр–Мур — трохи повільніший**
3. Рабін–Карп — найповільніший**

Висновок

Найефективнішим алгоритмом виявився КМП.
